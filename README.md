# Тестовое статическое веб-приложение

Одностраничный сайт для проверки вёрстки, типографики и встроенных медиа (изображения, SVG, видео, аудио). Собирается в чистый статический каталог без серверной части.

## Стек

- [Vite](https://vitejs.dev/) 6 — dev-сервер и production-сборка
- HTML, CSS, минимальный JavaScript (`src/main.js`)

## Возможности демо-страницы

- Разнообразный текст: абзацы, цитаты, списки, боковая колонка, тёмная секция, пример кода
- Фотографии разных пропорций — **локальные JPEG** в `public/media/images/`
- Векторная графика: инлайн SVG и файлы в `public/assets/`
- Два HTML5-видео и два аудиоэлемента — **локальные файлы** в `public/media/videos/` и `public/media/audio/`

После `npm run build` весь контент лежит в `dist/` (включая `dist/media/...`) и отдаётся с того же сервера, без внешних URL на картинки и медиа.

## Структура проекта

```
├── index.html          # точка входа
├── vite.config.js
├── public/
│   ├── assets/         # SVG (копируются в dist как есть)
│   └── media/          # изображения, видео, аудио → dist/media/
├── src/
│   ├── main.js
│   └── style.css
└── dist/               # результат `npm run build` (не коммитить)
```

## Требования

- [Node.js](https://nodejs.org/) с npm (рекомендуется актуальная LTS)

## Команды

| Команда        | Назначение                          |
| -------------- | ----------------------------------- |
| `npm install`  | установка зависимостей              |
| `npm run dev`  | режим разработки с hot reload       |
| `npm run build`| сборка в `dist/` (`base: './'`)     |
| `npm run preview` | локальный просмотр собранного сайта |

## Лицензия

Приватный тестовый проект; демо-медиа принадлежат их правообладателям и используются только как примеры встраивания.
